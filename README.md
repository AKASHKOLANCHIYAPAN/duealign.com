## ✨ Key Features & Capabilities

### 1. 🎯 Interactive 3-Step Project Estimator
- Instant cost & timeline calculator based on service type, project size, and delivery urgency.
- Customized pricing matrix calibrated in **Indian Rupees (₹)**:
  - **Websites**: ₹2k – ₹15k+
  - **Mobile Apps**: ₹8k – ₹50k+
  - **AI Automation**: ₹5k – ₹30k+
  - **UI/UX Design**: ₹2k – ₹15k+
  - **Content Creation**: ₹1.5k – ₹8k/mo
  - **Graphic Design**: ₹2k – ₹12k+

### 2. ⚡ Services & Core Offerings
1. **Website & Web App Development** (Business sites, landing pages, web apps, e-commerce)
2. **AI Tools & Workflow Automation** (Custom chatbots, task automation, AI content agents)
3. **UI/UX Design** (App UI, Figma prototypes, design systems)
4. **Content Creation** (Website copywriting, social media posts, articles, scripts)
5. **Graphic Design & Branding** (Logo design, brand kits, marketing assets)
6. **Dedicated Remote Team** (Monthly retainers, dedicated developers/designers)
7. **Mobile App Development** (Android, iOS, React Native apps)

### 3. 💼 Careers & Team Recruitment
- Dedicated **Careers / Join Us** section enabling developers, designers, AI engineers, and writers to submit resumes directly to `duealigntechsolution@gmail.com`.

### 4. 📬 Direct Email Contact Integration
- Contact form powered by **FormSubmit** delivering project inquiries straight to `duealigntechsolution@gmail.com` with budget selection, service requested, and project brief.

### 5. 📱 Seamless Mobile Responsiveness
- Glassmorphism sticky top navigation with touch-friendly mobile drawer menu.
- Fully responsive across desktop (1440px+), tablet, and mobile viewports.

## 🛠️ Technology Stack

- **HTML5**: Semantic, accessible structure with OpenGraph & Twitter Meta tags for SEO.
- **Vanilla CSS3**: Custom design tokens, CSS variables, glassmorphism, responsive grid & flex layouts.
- **Vanilla JavaScript (ES6+)**: Scroll progress bar, IntersectionObserver scroll-reveal animations, interactive estimator state machine, counter animations, and mobile nav drawer toggle.
- **Fonts**: Google Fonts (`Outfit`, `Space Grotesk`, `JetBrains Mono`).

## 💻 Running Locally

To preview and run the website locally:

```bash
# Clone the repository
git clone https://github.com/AKASHKOLANCHIYAPAN/duealign.com.git

# Navigate into the project folder
cd duealign.com

# Start a local HTTP server using Python
python -m http.server 3000
```

Open your browser and visit **[http://localhost:3000](http://localhost:3000)**.
